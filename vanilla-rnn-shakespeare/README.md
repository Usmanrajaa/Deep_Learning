# Vanilla RNN – Character‑Level Shakespeare Text Generation

**Character‑Level Shakespeare Text Generation with a From‑Scratch Vanilla RNN**  
Built a pure `tanh` RNN from first principles in PyTorch to model character‑level Shakespeare.  
Trained on 1.1M characters, reducing loss from **2.97 → 1.70**.  
Generated text shows local coherence and learned formatting; long‑range incoherence demonstrates **vanishing gradients**.  
Code modular, tested, and documented.
